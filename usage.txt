Usage of growisofs


"To continue a DVD (multisession)
  growisofs -M /dev/dvd -R -J /more/files

The software will automatically search for the last track
written to the DVD, and append the new track to it."
--axllent.org/


"To burn an .iso file to a DVD
  growisofs -Z /dev/dvd=file.iso

This will burn an .iso file to a DVD"
--axllent.org/
